# @isobel/nasa

The NASA service is an example of a minimum implementation of an Isobel service. It is available to guide you on creating your own services.
[View on GitHub](https://github.com/nathsimpson/isobel/tree/master/packages/services/nasa).

## Actions

### .fetchPhotoOfTheDay

Retrieves information on the NASA Photo of the fetchPhotoOfTheDay

```
{
  "url":"https://apod.nasa.gov/apod/image/....jpg",
  "hdurl":"https://apod.nasa.gov/apod/image/....jpg",
  "explanation":"Description goes here"}
```
