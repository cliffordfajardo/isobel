# @isobel/twitter

## Actions

### .fetchLatestTweets

Returns an array of your latest Tweets

#### Params

`username`: The username of the account to retrieve tweets from

## Getting your API Key

Head to [developer.twitter.com/apps](https://developer.twitter.com/apps), and create a new Twitter App. Once your app is created, click on 'Keys and Tokens', and paste all of the values in your .env file under the following keys...

## Expected env vars

- TWITTER_CONSUMER_KEY
- TWITTER_CONSUMER_SECRET
- TWITTER_ACCESS_TOKEN
- TWITTER_ACCESS_TOKEN_SECRET
